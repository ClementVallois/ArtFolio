## Configuration de Tailwinds sur le container docker 
- `docker exec artfolio-frontend-main-1 npm install -D tailwindcss postcss autoprefixer`
- `docker exec artfolio-frontend-main-1 npx tailwindcss init -p`
## Configuration de de vue-router sur le container docker 
- `docker exec artfolio-frontend-backoffice-1 npm install vue-router@4`
## Ajout de headlessui
- `npm install @headlessui/vue@latest`
